🌿 Gathering Information
Gathering areas (0x398): up to 2 Area Information blocks, each pointing into the Gathering Point Data table (0x310). Each Gathering Point has: X/Z/Y position, gathering code (index into point data), and frequency upper limit. Point Data entries hold per-item probability+item code tables, terminated by 0xFFFF.
